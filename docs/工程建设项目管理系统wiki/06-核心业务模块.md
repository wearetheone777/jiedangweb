# 核心业务模块

项目涵盖工程建设全生命周期管理，按业务领域分为以下核心模块。所有业务模块都遵循统一的 CRUD + 审批 + 批量导出模式。

## 1. 进度管理

### 施工日志 (`constructionLog`)

| 属性 | 内容 |
|------|------|
| API 端点 | `/api/construction_logs/` |
| 页面 | 列表 → 查看/新增/编辑 |
| 审批流 | 支持多级审批 |
| 特殊功能 | 批量导出图片 (`batch_export_images/`) |

## 模块共性

所有业务模块（除合同管理和问题整改外）都遵循统一的接口模式：

```
GET    /api/{resource}/          # 列表（分页）
GET    /api/{resource}/{id}/     # 详情
POST   /api/{resource}/          # 新增
PATCH  /api/{resource}/{id}/    # 更新
DELETE /api/{resource}/{id}/    # 删除
POST   /api/{resource}/{id}/approve/          # 审批
GET    /api/{resource}/{id}/approve_records/  # 审批记录
GET    /api/{resource}/batch_export_images/   # 批量导出图片
```

# 项目概览

## 基本信息

- **项目名称**：工程建设项目管理系统 (xmgl)
- **版本**：1.0.0 (versionCode: 101)
- **AppID**：`__UNI__13B505C`
- **项目类型**：uni-app 跨平台移动端应用（Vue 2）
- **运行平台**：Android / iOS（通过 HBuilderX 打包为原生 APP）

## 项目定位

本项目是一个面向工程建设领域的移动端 **项目全生命周期管理工具**，覆盖从开工准备到竣工投营的完整流程。系统支持建设单位（甲方）、施工单位、监理单位、设计单位等多角色协同工作，核心功能包括：

- **进度管理**：施工日志、监理日志、月报总结、竣工报告、投营报告
- **开工管理**：前期手续备案、开工报告
- **安全管理**：视频监控、三级培训、安全监督、班前教育
- **质量管理**：材料验收、旁站记录、隐蔽工程
- **投资管理**：变更签证
- **合同管理**：合同备案、合同变更
- **问题管理**：问题整改

## 业务角色

系统根据 `userInfo.dept_type` 区分为四种角色，不同角色在首页看到的功能模块不同：

| 角色类型 | 枚举值 | 说明 |
|---------|--------|------|
| 建设单位 | `org` | 甲方，拥有全部模块权限，含超管 |
| 施工单位 | `construction_unit` | 施工方，部分模块 |
| 监理单位 | `supervision_unit` | 监理方，与建设单位模块接近 |
| 设计单位 | `design_unit` | 设计方，仅投资管理模块 |

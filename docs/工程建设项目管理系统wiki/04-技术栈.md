## 关键依赖

### HTTP 客户端 — luch-request

位置：`js_sdk/luch-request/`

基于 axios 风格的请求库，提供：请求/响应拦截、全局配置、自定义 header 等功能。

### UI 组件库 — uni-ui

项目依赖 uni-app 官方组件库 `uni_modules/uni-ui`，使用了大量官方扩展组件：

| 组件 | 用途 |
|------|------|
| uni-forms + uni-easyinput | 表单输入与校验 |
| uni-popup | 弹窗/对话框 |
| uni-file-picker | 文件选择与上传 |
| uni-swipe-action | 列表滑动操作（编辑/删除） |
| uni-collapse | 折叠面板 |
| uni-steps | 步骤条（显示项目阶段） |
| uni-data-picker / uni-data-select | 数据选择器 |
| uni-list / uni-table | 列表与表格 |
| uni-load-more | 上拉加载更多 |
| uni-icons | 图标库 |
| uni-nav-bar / uni-search-bar | 导航与搜索 |
| uni-calendar / uni-datetime-picker | 日历/日期选择 |
| uni-card | 卡片容器 |
| uni-tag / uni-badge | 标签/徽标 |
| uni-segmented-control | 分段控制器 |
| uni-countdown | 倒计时 |
| uni-drawer | 侧边抽屉 |
| uni-fab | 悬浮按钮 |

### 图表 — lime-echart

位置：`uni_modules/lime-echart/`

基于 ECharts 的 uni-app 图表组件，用在首页展示统计图表。内置了 `echarts.min.js` 全量包。

项目中的图表组件：
- `barGraph` — 安全监督柱状图
- `pieMap` — 问题整改统计饼图
- `radarMap` — 项目过程统计雷达图

## 后端技术栈（推测）

根据 API 端点命名风格（RESTful、`/api/*/` 格式、JWT 认证），后端可能基于 **Django REST Framework (DRF)** 构建。关键特征：

- Token 认证端点：`api-token-auth/`
- 分页参数：`page` / `page_size`
- 支持 `GET/POST/PATCH/DELETE` 标准 REST 操作
- 自定义 action：`/approve/`、`/approve_records/`、`/to_rectify/` 等

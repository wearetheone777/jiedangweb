# 关键类与函数

## Vue.prototype 全局方法

### `canApprove(data)`
- **位置**：`main.js:8-13`
- **功能**：判断当前用户是否可以审批指定单据
- **参数**：`data` — 单据对象（需包含 `state` 和 `cur_approvers` 字段）
- **返回值**：`Boolean`

### `verifyPermission(name)`
- **位置**：`main.js:21-26`
- **功能**：校验当前用户是否具有指定权限
- **参数**：`name` — 权限名称字符串，多个用逗号分隔（或逻辑）
- **返回值**：`Boolean`

### `preview(url, urls)`
- **位置**：`main.js:28-33`
- **功能**：预览图片
- **参数**：`url` — 当前图片URL，`urls` — 所有图片URL数组

### `downLoadFile(url)`
- **位置**：`main.js:34-71`
- **功能**：下载并打开文件（下载 → 保存 → 打开文档）

## 自定义组件

### `file-update` — 文件上传

**生命周期回调**：
- `select(e)` — 选择文件后自动上传到服务器，通过 `@add` 事件返回上传结果
- `remove(e)` — 删除已上传文件，通过 `@remove` 事件通知父组件
- `initFileList(val)` — 父组件调用，初始化已存在的文件列表

### `time-line` — 审批时间线

**核心渲染逻辑**：
- 遍历 `log` 数组，每条记录生成一个节点
- 每个节点包含：图标（同意/驳回）、审批人、结果、节点名称、时间、备注
- 最后一条节点不显示连接线（`v-if="id != log.length - 1"`）

### 图表组件 (`barGraph` / `pieMap` / `radarMap`)

**初始化流程**：
```
mounted → this.$refs.chart.init(echarts, callback)
                                   ↓
                          chart.setOption(this.option)
                                   ↓
                          (可选) chart.resize({width, height})
```

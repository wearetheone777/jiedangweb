## 页面通用模式

所有业务页面遵循统一的 CRUD 模式：

1. **列表页**（如 `constructionLog/constructionLog.vue`）
   - 顶部搜索/筛选栏 → 状态、项目、日期等
   - 中部列表 → 使用 `uni-swipe-action` 支持左滑编辑/删除
   - 底部加载更多 → `onReachBottom` 事件触发分页

2. **新增/编辑页**（如 `constructionLog/add.vue`）
   - `uni-forms` 表单 + `uni-easyinput` 输入
   - `file-update` 组件用于文件上传
   - `project-picker`/`org-picker` 用于选择关联数据

3. **详情/查看页**（如 `constructionLog/viewLog/viewLog.vue`）
   - 展示完整业务数据
   - `time-line` 组件展示审批进度
   - 审批按钮（如果当前用户是审批人）

4. **审核页**（如 `completionReport/check/check.vue`）
   - 详情展示 + 审批操作（通过/驳回）
   - `visa-time-line` 显示审批历史

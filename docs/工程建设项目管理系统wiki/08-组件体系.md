# 组件体系

## 自定义业务组件

### file-update — 文件上传组件

**位置**：`components/file-update/file-update.vue`

封装 `uni-file-picker`，支持图片和文件选择和自动上传到后端。

```
Props:
  media: Boolean     — true=仅图片, false=所有文件
  belong_to: String  — 上传时文件归属分类（如 "用户/头像"）

Events:
  @add({id, url})    — 上传成功回调
  @remove({id})      — 删除文件回调

Methods:
  initFileList(val)  — 初始化已上传文件列表
```

上传流程：选择文件 → uni.uploadFile → 后端返回文件 URL → 通过 `@add` 事件回传父组件。

### time-line — 审批时间线组件

**位置**：`components/time-line/time-line.vue`

展示审批记录的纵向时间线，每条记录包含审批人、结果（同意/驳回）、节点名称、时间、备注。

```
Props:
  log: Array     — 审批记录数组 [{approver, result, node_name, created_time, remark}]
  empty: String  — 空数据提示文字，默认"待审批"
```

- 同意显示 ✅ 图标（蓝色），驳回显示 ❌ 图标（红色）
- 最后一条记录不显示连接线

### visa-time-line — 变更签证时间线组件

**位置**：`components/visa-time-line/visa-time-line.vue`

与 time-line 类似，但适配变更签证的数据结构（使用 `org` / `role` / `data` 字段）。

```
Props:
  log: Array — [{org, role, data, result}]
```

### 图表组件

三个基于 `lime-echart` 封装的图表组件，用于首页数据看板：

| 组件 | 图表类型 | 标题 |
|------|---------|------|
| `barGraph` | ECharts 柱状图 | 安全监督及培训教育 |
| `pieMap` | ECharts 饼图（环形） | 问题整改统计 |
| `radarMap` | ECharts 雷达图 | 项目过程统计 |

> **注意**：图表数据目前为静态 Mock 数据，未接入真实 API。各组件在 `mounted` 中调用 `chart.setOption(this.option)` 渲染。

### 选择器组件

#### project-picker — 项目选择器

**位置**：`components/project-picker/project-picker.vue`

```
Props:
  deptType: String     — 选择类型 (org/supervision/construction/design)
  autoInit: Boolean    — 是否自动加载选项
  noName: Boolean      — 是否隐藏标签名
  immediately: Boolean — 是否立即返回
  initQuery: Object    — 初始化查询参数
```

支持不同角色获取对应的项目列表，通过 `api/common.js` 的 `getCommonList` 获取数据。

#### org-picker — 单位选择器

**位置**：`components/org-picker/org-picker.vue`

```
Props:
  deptType: String     — 单位类型 (org/supervision/construction/design/audit)
  labelName: String    — 标签文字，默认"单位名称："
  autoInit: Boolean    — 是否自动加载
```

使用 `urlMap` 映射不同的 API 端点：

```javascript
urlMap = {
  org: '/api/orgs/?has_page=true',
  supervision: '/api/supervision_units/',
  construction: '/api/construction_units/',
  design: '/api/design_units/',
  audit: '/api/audit_units/',
}
```

#### person-picker — 人员选择器

**位置**：`components/person-picker/person-picker.vue`

选择具体人员（如移交人），通过 `api/common.js` 获取人员列表。

#### momo-multipleSelect — 底部弹出多选组件

**位置**：`components/momo-multipleSelect/momo-multipleSelect.vue`

从底部弹出的多功能选择器，支持：

- 单选/多选模式
- 全选/取消全选
- 禁用选项
- 默认选中值
- 蒙层点击关闭
- 底部弹出动画

```
Props:
  value: Boolean        — 显示/隐藏
  data: Array           — 数据源 [{label, value, disabled}]
  labelName: String     — label 字段名 (默认 "label")
  valueName: String     — value 字段名 (默认 "value")
  mode: String          — "multiple" (多选) / "single" (单选)
  defaultSelected: Array — 默认选中值
  allShow: Boolean      — 是否显示全选按钮
  maskCloseAble: Boolean — 是否允许点击蒙层关闭

Events:
  @confirm(selectedData) — 确认选择
  @cancel()              — 取消选择
```

## 第三方 UI 组件库 (uni_modules)

项目引入了大量的 uni-ui 扩展组件，详见 [目录结构 -> uni_modules](03-目录结构.md)。核心依赖组件：

- **uni-forms**：表单校验，支持自定义校验规则
- **uni-popup**：弹窗/对话框，用于修改密码等
- **uni-file-picker**：文件选择上传
- **uni-swipe-action**：列表滑动操作
- **uni-collapse**：项目详情的参与单位展示
- **uni-easyinput**：增强输入框，支持前后插槽
- **uni-data-picker/uni-data-select**：数据选择
- **uni-steps**：项目阶段进度条
- **uni-table**：数据表格展示
